# KR100ToAppleMusic
국내 음원 사이트 Top100 음원 기반 AppleMusic 플레이리스트 생성 프로젝트

## 활용 기술
- Swift
- MVC Pattern
- Apple Music API
- Alamofire
- SwiftSoup
- Firebase
- SwiftJWT
